盒模型

2019年 10月 by Alex

盒 模型 是 HTML / CSS 的基石

  • 所有的标签都是一个盒子
  • 盒子 的高度和宽度由 margin / border / padding / content 共同决定

box-sizing 属性可以用来改变 盒模型 宽度高度的计算规则

display 属性可以用来改变 常见的 盒模型 的 显示模式 亦即 在 文档流中 显示规则

布局/定位

2019年 11月 by Alex

布局与定位 是指 如何 摆放 和 调整 盒模型的位置

  • 浮动 是 flex 之前最主要的 布局定位手段，也是网格布局的基础
  • 相对 / 绝对 / 固定 定位 定义了 兄弟标签和父子标签之间的位置如何进行调整
  • flex 简化了 如何在一个容器内如何 布局 元素

文本

2019年 11月 by Alex

文本是 Web 传递信息的主要载体

  • 文本 的常见 大小单位 是 pm / em / %
  • 文本类 属性 包括 对齐，装饰，缩进，阴影，变形与间距
  • 字体类 属性 包括 字体家族，字体大小，字体样式，字体变体，字体粗细

图片/背景

2019年 11月 by Alex

图片可以作为 前景 与 背景 作为背景时显示模式变得丰富多样

  • 前景图片 我们需要注意的显示像素尺寸 和 实际像素尺寸的区别
  • 背景图片 我们关注的是图片的 重复，背景位置，背景尺寸，背景附件和背景基准
